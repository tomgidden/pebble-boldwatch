THIS PROJECT IS NO LONGER SUPPORTED OR WORKING

I no longer use my Pebble watch, so my watchface repos are unmaintained and certainly fail on newer Pebble releases. You are welcome to fork this code and get it working again. Sorry!

Boldwatch for Pebble OS 1

Boldwatch comes in several different builds for the various options. Visit the list of builds on your phone (or scan the QR code below); select the appropriate ".pbw" file; and then click "Raw" to download that build. You can install more than one build at once.

Boldwatch for Pebble OS 2

The new Pebble OS and app allows the watchface to be reconfigured from the app itself, so only one build is needed. Click here on your phone (or scan the QR code below) to install it. Then, in the Pebble App's list of watchfaces, you should be able to click the round button to reconfigure it.

If you have any comments or bug reports, please file an issue or comment on GitHub, or message me on Twitter (@tomgidden)

Known bugs/issues

  • All versions: enabling the second-hand or display will exhaust the battery quicker. This is a property of the Pebble hardware which cannot be changed.
